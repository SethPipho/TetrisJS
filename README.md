#TetrisJS

**TetrisJs is a full featured Tetris Clone written in Javascript.** 

TetrisJS follows all(most) of the official rules/features in the [Official Tetris Guidlines](https://tetris.wiki/Main_Page) including:  
* SRS rotation  
* DAS  
* Wallkicks  
* Ghost Piece  
* Soft and Hard Drops  
* LockDelay
